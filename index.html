What is Ruby.

Ruby as any other object oriented programming language, resembles a natural world everything is an Object and there are integers, characters, text, arrays - everything. You put yourself in the place of an object and then do stuff related to other objects or oneself.

'self' defines the code to specify who you are., if we do not specify the class the default is 'main'.

Operating with objects: Ruby uses period to call any method over any object and the resultant is again another object.

Ruby shows us all the methods if we just need to check them, typing Object.methods returns all the methods separated by ':'
